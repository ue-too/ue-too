# Getting Started

`@ue-too/board-game-engine` provides game engine integration for building board-based games with the board camera system.

## Installation

```bash
npm install @ue-too/board-game-engine @ue-too/board
```

## Basic Usage

```typescript
import { GameEngine } from "@ue-too/board-game-engine";

const engine = new GameEngine();
engine.start();
```
